# Flappy-Bird-pygame

## Overview

A simple recreation of the classic "Flappy Bird" game developed using Python and Pygame. Control the bird, fly through randomly generated pipes, and achieve the highest score while avoiding collisions.


## Features

- Gravity-based bird movement
- Random pipe generation
- Collision detection
- Score tracking
- Start screen
- Game Over screen
- Restart functionality


## Technologies

- Python
- Pygame



## Controls

 **Key - Action**
 Space / Up Arrow - Jump 
 Left Mouse Button - Jump
   R  -   Reset Game
